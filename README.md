Here samples for scripts to extend Mysql Enterprise monitoring provided by Oracle Enteprrise Manager 24ai with metric extensions.

The script script_centralized_monitoring.pl is a perl sample that

. query Router REST API to check teh status opf all the routers
. print out a table summary with routerid, router address, router port, router status, router hostname, router uptime
